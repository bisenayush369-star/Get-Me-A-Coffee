# ☕ Get Me a Coffee

🔗 **Live Project:**  https://get-me-a-coffee.netlify.app/

A full-stack **Next.js** application that allows creators to receive support (donations) using **Razorpay**.

Built to understand real-world authentication, payments, and server-side logic in a production-style app.

---

## 🚀 Features

- 🔐 GitHub Authentication with NextAuth
- 💳 Razorpay payment integration
- 👤 User-based creator pages
- 📊 Dashboard to track received payments
- 🧾 Stores users & transactions securely in MongoDB
- 🔄 Server-side API routes using Next.js App Router
- ⚡ Fast performance with server components
- 🎨 Clean, responsive UI with Tailwind CSS
- 🔒 Secure server-side payment handling
- 🧠 Proper session & redirect handling

---

## 🛠️ Tech Stack

- **Next.js (App Router)**
- **React**
- **NextAuth**
- **Razorpay**
- **MongoDB**
- **Tailwind CSS**

---

## 🏗️ Architecture

- Single **full-stack Next.js application**
- API routes handled via `/app/api/*`
- No separate backend server required
- Authentication & payments handled securely on the server

## 📚 My Learnings

- Worked with **MongoDB** to store users and transaction data
- Learned when and why **API base URLs are not required** in Next.js full-stack apps
- Debugged real-world issues like redirect failures and session-based UI rendering
- Gained experience deploying **production-ready applications on Vercel**
- Improved understanding of **clean architecture, security, and scalability**
